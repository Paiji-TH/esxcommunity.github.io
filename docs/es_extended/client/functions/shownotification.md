# ESX.ShowNotification

This function shows a basic notification to the player.

#### Syntax

```lua
ESX.ShowNotification(text)
```

##### Required arguments
- **text**: A string 

#### Examples

##### Lua
```lua
ESX.ShowNotification('Hello')
```

#### Supported Color Syntax

* `~r~` --> Red
* `~b~` --> Blue
* `~g~` --> Green
* `~y~` --> Yellow
* `~p~` --> Purple
* `~o~` --> Orange
* `~c~` --> Grey
* `~m~` --> Dark Grey
* `~u~` --> Black
* `~n~` --> New Line
* `~s~` --> White (**Default**)
* `~w~` --> White
* `~h~` --> Bold
* `~nrt~` --> ?
